Privacy Policy

Last Updated: August 2, 2025

At CoreVitals, protecting our users' privacy is our top priority. This privacy policy explains how our application works and how we protect your data.

1. Information We Collect

CoreVitals application reads and displays your Mac computer's system information locally:

  • Network connection information (IP address, connection status)
  • Disk usage information (free space, total capacity)
  • Graphics card information (model, memory usage)
  • Running process information (CPU and memory usage)
  • Power and battery status information
  • System temperature and fan speed information

2. Data Collection and Usage

Important: CoreVitals application works completely locally. No personal data, system information, or usage data is sent to external servers.

  • All data is processed only on your device
  • No internet connection required
  • No data collection or analytics services used
  • No integration with advertising networks

3. Data Storage

The application does not permanently store any data except settings and preferences:

  • System information is read and displayed in real-time
  • No historical data is stored
  • Only user preferences (theme, display settings) are stored locally
  • All data is completely removed when the application is deleted

4. Data Security

Security measures we take to protect your data:

  • All data is processed in Apple's sandboxed environment
  • System information accessed only with necessary permissions
  • Data encryption and secure coding practices
  • Regular security updates

5. Third-Party Services

CoreVitals application:

  • Does not use any third-party analytics services
  • Contains no social media integration
  • Has no connection to advertising networks
  • Makes no external API calls

6. Children's Privacy

Our application does not knowingly collect data from children under 13 years of age. Parents can monitor and restrict their children's use of the application as needed.

7. User Rights

As CoreVitals users, your rights include:

  • You can remove the application at any time
  • You can manage system permissions from macOS settings
  • You can ask questions about the privacy policy
  • You can request transparency regarding data processing

8. Policy Changes

Changes to this privacy policy:

  • Will be announced through App Store application updates
  • Significant changes will be notified to users
  • The current version will always be published on this page
